# Sistema de Análise de Temperaturas e Recomendação de Roupas

Este projeto em **Java** analisa as temperaturas de diferentes cidades, compara com a média anual e fornece sugestões de roupas adequadas de acordo com o clima.

## Funcionalidades

- Exibe a temperatura atual de cada cidade.
- Compara a temperatura atual com a média anual da cidade.
- Indica se o dia está mais quente, mais frio ou igual à média anual.
- Sugere roupas apropriadas para o clima:
  - Temperaturas abaixo de 15°C → casaco.
  - Temperaturas entre 15°C e 25°C → roupas leves.
  - Temperaturas acima de 25°C → camiseta e shorts.
- Calcula a média atual geral e a média anual geral das cidades analisadas.

## Tecnologias

- **Java** – linguagem utilizada para implementar o sistema.
